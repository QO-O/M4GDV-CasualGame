# M4GDV-CasualGame

Chosen difficulty:
My pick: Extra Fun (simple input, clear goal, score/timer, at least 2 forms of feedback, rising difficulty, combo/multiplier, better feedback)

Core mechanic
My core mechanic is that the player needs to cut fabric to size and shape of the request.

Input
The player does this via dragging across the screen.

Goal
The player tries to make a whole set of clothing in the allocated time.

Easy to learn
The player learns the goal via the cutting lines, a scissor, and a small scissor icon next to the cut line.

Hard to master
The challenge comes from blunt scissors, no path correction, and the timer.

Why does this remain fun?
It's something you don't really think about, just do it.

Juiciness
I am making this satisfying via sparkles, a rising bar if you stay on the line, and commentary in-game as if the player is commenting to themself.
